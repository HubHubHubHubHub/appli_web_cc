# Raccourcis clavier

## Raccourcis globaux

| Touche   | Action                         | Contexte                 |
| -------- | ------------------------------ | ------------------------ |
| `Escape` | Dépingler la sidebar grammaire | Quand un mot est épinglé |

## Interactions texte interactif (mots `.ukr`)

| Action             | Effet                                                              |
| ------------------ | ------------------------------------------------------------------ |
| Survol souris      | Affiche l'infobulle + la table de grammaire dans la sidebar droite |
| Clic               | Épingle la sidebar grammaire (reste affichée après le survol)      |
| `Enter` / `Espace` | Même effet que le clic (accessibilité clavier)                     |

## Navigation sidebar

| Touche    | Action                              | Contexte                         |
| --------- | ----------------------------------- | -------------------------------- |
| `↑` / `↓` | Naviguer entre les mots de la liste | Focus dans la sidebar            |
| `Enter`   | Sélectionner le mot focalisé        | Focus sur un mot dans la sidebar |
